📌 Public Complaint Prioritization System
A smart web-based system that analyzes and prioritizes public complaints using intelligent techniques to ensure faster and more efficient resolution.
🔗 Live Demo: https://public-complaint-prioritization-system.onrender.com/

🚀 Features
📥 Submit public complaints easily
🧠 Intelligent prioritization based on severity and keywords
⚡ Faster identification of critical issues
📊 Organized complaint management system
🌐 Simple and user-friendly interface

🛠️ Tech Stack
Frontend: HTML, CSS, JavaScript
Backend: Python (Flask / FastAPI if used)
Database: (Add if used – e.g., SQLite / MongoDB)
Deployment: Render

⚙️ How It Works
User submits a complaint through the web interface
The system processes the complaint text
Priority is assigned based on predefined logic / ML model
Complaints are categorized and displayed accordingly
